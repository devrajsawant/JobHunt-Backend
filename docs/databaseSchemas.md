JOBS TABLE SCHEMA

| Field            | Type     | Required       | Description                           |
| ---------------- | -------- | -------------- | ------------------------------------- |
| _id              | ObjectId | auto           | Primary key generated by MongoDB      |
| title            | String   | Yes            | Job title                             |
| description      | String   | Yes            | Full job description                  |
| skills           | String[] | Yes            | Required skills                       |
| location         | String   | Yes            | Job location                          |
| salary           | String   | No             | Salary range (ex: "10-15 LPA")        |
| experience       | String   | No             | Required experience (ex: "2-4 years") |
| workMode         | String   | Yes            | remote / hybrid / onsite              |
| jobType          | String   | No             | full-time / internship / contract     |
| companyId        | ObjectId | Yes            | Reference to Company collection       |
| applicationCount | Number   | default 0      | Total applications received           |
| status           | String   | default "open" | open / closed / paused                |
| expiresAt        | Date     | No             | Job expiry date                       |
| createdAt        | Date     | auto           | Job creation timestamp                |
| updatedAt        | Date     | auto           | Last update timestamp                 |

COMPANY TABLE SCHEMA

| Field           | Type     | Required    | Description                                         |
| --------------- | -------- | ----------- | --------------------------------------------------- |
| _id             | ObjectId | auto        | Unique identifier generated by MongoDB              |
| name            | String   | Yes         | Name of the company                                 |
| description     | String   | Yes         | Detailed description about the company              |
| industry        | String   | Yes         | Industry category (e.g., Tech, Finance, Healthcare) |
| size            | String   | Yes         | Employee count range (e.g., 1-10, 10-50, 50-200)    |
| location        | String   | Yes         | Primary location of the company                     |
| logo            | String   | Yes         | URL of the company logo                             |
| website         | String   | No          | Official company website                            |
| contactEmail    | String   | No          | Company contact email                               |
| contactPhone    | String   | No          | Company phone number                                |
| linkedin        | String   | No          | LinkedIn company page                               |
| twitter         | String   | No          | Twitter/X handle                                    |
| ownerId         | ObjectId | Yes         | Reference to the user who owns the company          |
| rating          | Number   | Default 0   | Average company rating                              |
| reviewCount     | Number   | Default 0   | Total number of reviews                             |
| totalJobsPosted | Number   | Default 0   | Total jobs posted by the company                    |
| slug            | String   | Yes         | URL-friendly identifier (e.g., google, netflix)     |
| createdAt       | Date     | auto        | Timestamp when company was created                  |
| updatedAt       | Date     | auto        | Timestamp when company was last updated             |
